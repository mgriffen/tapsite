What You Can Extract

Design Intelligence Toolkit

From pixel-perfect screenshots to complete design systems. Ask your AI to extract — tapsite handles the rest, returning structured data ready to use.

Browse & Interact
Navigate any site — even behind MFA — click elements, scroll, and interact. Your AI gets the same access you have in your browser.
Visual Capture
Full-page screenshots, individual elements, every image and SVG on the page. Download assets to disk — ready for your design files or codebase.
Design System Extraction
Pull the complete design language from any site: colors with usage frequency, full type scale, spacing system, CSS variables, breakpoints, and animation patterns. Export as W3C design tokens.
Content & Structure
Extract clean markdown content, page metadata, every link mapped and classified, table data structured, and form fields inventoried. Understand any page at a glance.
Network & API Discovery
See what APIs a site calls, auto-generate OpenAPI schemas from observed traffic, and detect the full tech stack. Great for competitive research and integration planning.
Performance & Accessibility
Instant WCAG accessibility audits with scored results. Core Web Vitals, resource budgets, dark mode detection. Catch issues before your users do.
Compound Tools

One Command. Everything You Need.

Workflow tools chain multiple extractors into a single call. Run a complete audit, extract a full design system, or inventory an entire site — all from one prompt.

✓
tapsite_audit
Pre-launch quality check with scored report. Chains: a11y + contrast + perf + metadata + dark mode + forms.
▨
tapsite_designsystem
Full design system extraction with W3C token export. Chains: colors + fonts + spacing + shadows + css_vars + breakpoints + animations + icons + components.
⚔
tapsite_teardown
Competitive design teardown — all extractors in one call. Colors, fonts, spacing, shadows, components, breakpoints, animations, icons, stack, perf, a11y, contrast, dark mode.
⇄
tapsite_harvest
Migration asset inventory. Crawls up to N pages, extracting content, images, SVGs, forms, fonts, and links per page.

In Action

From Prompt to Design System

Tell your AI what you need. tapsite handles the login, navigation, and extraction behind the scenes.

tapsite — mcp session
// Login once — your credentials stay in the browser, never in the chat
> tapsite_login_manual
✓ Browser opened. Login normally (MFA works too)...
✓ Session saved. You won't need to login again.

// Now extract a competitor's entire design system
> tapsite_extract_colors { url: "https://competitor.com" }
  colors: 42 unique | palette: #1a1a2e, #16213e, #0f3460, #e94560...

> tapsite_extract_fonts
  families: 3 | sizes: 8 | weights: 4

> tapsite_export_design_report
✓ Design report ready: output/design-report-2026/
  report.html | design-tokens.json | design-tokens.css
