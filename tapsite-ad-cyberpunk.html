Core Capabilities

Web Intelligence Toolkit

From pixel-perfect screenshots to full design system extraction. Every tool returns structured, token-efficient data optimized for AI consumption.

Browse & Interact
Navigate authenticated dashboards, click elements by index, scroll, and execute custom JavaScript. Full browser automation behind MFA-protected walls.
Visual Capture
Full-page and element-targeted screenshots. Download all images, extract inline SVGs, pull favicons. Capture the visual layer completely.
Design System Extraction
Colors, fonts, CSS variables, spacing scales, breakpoints, and component patterns. Export as W3C design tokens or full HTML design reports.
Content & Structure
Extract readable markdown content, metadata, link maps, table data, and form structures. Compressed DOM with indexed interactive elements.
Network Intelligence
Capture live network traffic with body inspection. Detect API endpoints and generate OpenAPI schemas. Filter by method, URL, or resource type.
Performance & A11y
Web Vitals, resource analysis, DOM complexity. WCAG AA/AAA accessibility audits with scored results. Dark mode detection and palette capture.

In Action

From Prompt to Data

Tell Claude what you need. tapsite handles authentication, navigation, and extraction behind the scenes.

tapsite — mcp session
// Login with MFA — credentials never touch the chat
> tapsite_login_manual
✓ Browser opened. Complete login & MFA in the window...
✓ Session saved to persistent profile

// Extract the design system from an internal dashboard
> tapsite_extract_colors { url: "https://internal.app/dashboard" }

> tapsite_extract_fonts

> tapsite_export_design_report
